Character Routes
Endpoint Description
ALL /api/v1/characters?page=${number}&perPage=${number} get all characters
SEARCH /api/v1/characters?search=${query} search all characters by name
ID /api/v1/characters/${id} get a character by id
RANDOM /api/v1/characters/random?count=${number} get random characters
MARIO /api/v1/characters/mario?page=${number}&perPage=${number} get mario characters
ZELDA /api/v1/characters/zelda?page=${number}&perPage=${number} get zelda characters
STAR FOX /api/v1/characters/star_fox?page=${number}&perPage=${number} get star fox characters
METROID /api/v1/characters/metroid?page=${number}&perPage=${number} get metroid characters
KIRBY /api/v1/characters/kirby?page=${number}&perPage=${number} get kirby characters
ANIMAL CROSSING /api/v1/characters/animal_crossing?page=${number}&perPage=${number} get animal crossing characters
DONKEY KONG /api/v1/characters/donkey_kong?page=${number}&perPage=${number} get donkey kong characters
F-ZERO /api/v1/characters/f-zero?page=${number}&perPage=${number} get f-zero characters
SPLATOON /api/v1/characters/splatoon?page=${number}&perPage=${number} get splatoon characters
MOTHER /api/v1/characters/mother?page=${number}&perPage=${number} get mother characters
